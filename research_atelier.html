个人主页执行总控台科研面板跳槽作战板申请规划

把论文、复现、结果与心得，收成一个真正能长期沉淀的研究工作台。

这里把科研全流程拆成四块：Paper Library、Research Notebook、Reproduction Board 和 Reflection Journal。PDF 最适合以 GitHub 仓库相对路径或在线链接的形式登记在这里，真正的 PDF 文件可以放在仓库的 papers 目录里。

论文库复现项目专业学习笔记

Paper Library

记录论文标题、学科、分支、PDF 路径、代码路径、状态和你的三句话摘要。

Research Notebook

围绕问题、公式、实验和卡点写结构化笔记。

Reproduction Board

把复现代码、数据、结果图、指标和结论摆清楚。

Reflection Journal

记录你自己的判断：值不值得做、难点是什么、下一步怎么推进。

和原站点共用同一份本地存储；可公开的内容上线到 GitHub Pages，私密内容继续保留在本地。
